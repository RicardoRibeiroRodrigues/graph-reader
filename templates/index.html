Graph2csv

Welcome to our project! This project allows you to convert graphs into tables (CSV format), divided between synthetic and hand-drawn.

Synthetic Graph Handdrawn Graph

How to Use Synthetic Graph

First, select the option above, synthetic graph. Then, upload the file using a button. When the graph appears, select the graph area, the X-axis, and the Y-axis by dragging the mouse. Finally, click on 'submit'.

How to Use Handdrawn Graph

First, set the threshold, blur, and K. Then, select the graph and send it! It's simple!
