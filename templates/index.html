Graph2csv

Welcome to our project! This project allows you to convert graphs into tables (CSV format), divided between synthetic and hand-drawn. Project made for the discipline Computer Vision (Prof. MARCELO HASHIMOTO) at Insper. Authors: Guilherme Fontana, Gustavo Guedes and Ricardo Ribeiro.

How to Use Synthetic Graph

First, select the option above, synthetic graph. Then, upload the file using a button. When the graph appears, select the graph area, the X-axis, and the Y-axis by dragging the mouse. Finally, click on 'submit'.

How to Use Handdrawn Graph

First, select the parameters in the configuration tab. Then, try to send the graph. If the displayed result is poor, review the parameters and try again until you get the desired outcome.
